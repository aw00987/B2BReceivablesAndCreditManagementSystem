# 技術選型

## バックエンド
- **言語/基盤**: Java 17 / Spring Boot 3.x
- **ビルド管理**: Maven
- **データベース**: MySQL 8.0
- **ORM**: Spring Data JPA (Hibernate)
- **認証・認可**: Spring Security + JWT
- **ライブラリ**: Lombok

## フロントエンド
- **言語/基盤**: TypeScript / React 18
- **ビルドツール**: Vite
- **UIコンポーネント**: Ant Design 5
- **スタイリング**: Tailwind CSS
- **ルーティング**: React Router v6
- **データ取得**: React Query
- **状態管理**: Zustand
- **HTTPクライアント**: Axios

## インフラ・デプロイ
- **インフラ**: AWS EC2
- **コンテナ化**: Docker / Docker Compose
- **構成**: Frontend (Nginx) + Backend (Spring Boot) + MySQL

## 開発ツール
- **VCS**: Git / GitHub
- **CI/CD**: GitHub Actions
- **監視**: Spring Boot Actuator
